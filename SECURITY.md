# Security Policy

## Reporting a Vulnerability
Please report security issues privately.

- Open a GitHub Security Advisory (preferred), or
- Email the maintainer listed in the repository profile.

## Scope
This project runs untrusted code in sandboxed Kubernetes Jobs.
If you find:
- sandbox escape
- secrets leakage
- policy bypass
- network policy bypass

## Responsible Disclosure
Do not publish exploit details until a fix is released.
