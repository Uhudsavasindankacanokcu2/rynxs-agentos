# Rynxs Roadmap

This document outlines the planned trajectory for the Rynxs platform.

---

## Phase 3: Travel & Sessions

- **Universal Travel**: Cross-universe travel sessions with filtered memory bridging.
- **Session Lifecycle**: Ephemeral workspace provisioning for risky tool execution.

---

## Infrastructure & Security

- **Hardened Sandboxes**: Native integration with gVisor and Kata Containers.
- **Policy-as-Code**: Integration with OPA Gatekeeper or Kyverno for manifest-level governance.
- **Multi-Channel Inbox**: WebSocket and gRPC gateways for distributed event sourcing.

---

## Observability

- **SIEM Link**: Automated audit log ingestion via OpenTelemetry or Fluent Bit.
- **Visual Control Plane**: Dashboard for fleet management and audit analysis.
